# Music Chord Finder

A small Python command-line tool that identifies basic chords from note names.

## Current Features

- Major chords
- Minor chords

## Upcoming Features
- Diminished chords
- Augmented chords
- Inversion support
- Sharp notes support

## Example

Input:
C E G

Output:
C major

## How to Run

### For Mac
python3 chord_finder.py

### For Windows
python chord_finder.py
